KRemote Files is a set of APIs for executing all sorts of operations on files between an Android device or a PC and a remote server through HTTP: file upload/download, access of remote files and directories, call of remote Java methods.

KRemote Files includes zero-learning APIs to access remote files, upload or download files:

  • The RemoteFile class has the same methods names and signatures as java.io.File: a RemoteFile method is a File method that is executed on the remote host.
  • The RemoteInputStream class extends InputStream and allows easy file download.
  • The RemoteOutputStream class extends OutputStream and allows easy file upload.
